# LIKUO_DB — 立國公司專案資料庫

本專案用於集中管理立國公司相關的資料庫設計、分析產出與交付文件。

## 目錄說明

| 目錄 | 用途 |
|------|------|
| `00_templates/` | 範本檔案（PRD、交付、維運） |
| `01_Discovery/` | 需求探索與訪談紀錄 |
| `02_Analysis/` | 分析產出（模型、規格、報告） |
| `03_Solution/` | 工程規格、系統移交摘要 |
| `docs/` | 對外交付文件與操作手冊 |
| `docs/prototype/` | 雛型 HTML 與操作說明 |
| `internal_docs/` | 內部專案管理文件 |
| `90_delivery_packages/` | 交付包根目錄，存放正式版本交付物 |
| `99_Trace/` | 追溯用原始文件（不修改、僅供引用） |
