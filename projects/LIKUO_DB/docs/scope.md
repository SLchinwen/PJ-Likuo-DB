# 系統範圍與階段規劃

## MCR 工程維護管理系統

### 第一階段（已完成）
- 廠區、設備、型號工序、合約、案件建檔
- 現場回報、審核、成果報表、進度追蹤

### 第二階段（進行中）
- 工程回報拍照、請款結算
- 基本資料批次匯入
- 吊車排程（暫停中）

## HR 出缺勤系統（暫停中）
- 刷卡匯入與出勤判斷
- 點名輔助、月結算、法規稽核
