    第一階段操作手冊_標準版（MCR）

    • 日期： 2026-01-29
    • 版本： v1.0（標準化整理）
    • 範圍： 第一階段已完成之 MCR 模組操作說明

    1. 文件目的

    將第一階段操作手冊以一致格式整理，作為對外交付與內部培訓基準版本。

    2. 共通前置條件

    1. 使用者已取得系統帳號並完成登入。
    2. 若為案件/回報相關功能：需先完成廠區、設備、型號工序、案件等基礎資料建置。
    3. 若需上傳照片：需確認可使用相機/檔案存取權限，並符合現場規範。

    3. 共通操作結構

    • 查詢區：輸入篩選條件（案件/廠區/設備/日期等）
    • 清單區：顯示查詢結果與待處理項目
    • 明細區：顯示單筆資料內容、工序與照片
    • 功能按鈕：新增、儲存、回報、退回、核准、下載等

    實際可見欄位與按鈕依帳號權限顯示。

    4. 共通常見錯誤與注意事項

    4.1 常見錯誤

    • 查詢無資料：請確認案件/日期/設備條件是否正確。
    • 無法送出或儲存：請檢查必填欄位是否完整。
    • 上傳失敗：請檢查網路、檔案大小或權限設定。
    • 審核退回：常見原因為照片不清晰、角度不符、回報工序錯誤。

    4.2 注意事項

    • 建議以「即時回報」為原則，避免跨日集中回報造成時序爭議。
    • 照片需可辨識設備與成果，必要時拍攝關鍵特寫。
    • 若需新增非標準工序，請依治理規範先由管理者新增後再回報。

    MCR100｜廠區資料建檔

    • 對應畫面： https://s3.hicloud.net.tw/leekuo/HTML/image/mcr100.png
    • 功能目的： 建立與維護廠區主檔，作為後續案件與回報依據。

    標準操作步驟

    1. 由主選單進入「廠區資料建檔」。
    2. 於查詢區輸入條件後點選「查詢」。
    3. 新增或編輯資料，點選「儲存」完成。

    MCR110｜設備資料建檔

    • 對應畫面： https://s3.hicloud.net.tw/leekuo/HTML/image/mcr110.png
    • 功能目的： 建立與維護設備主檔，供案件管理與回報使用。

    標準操作步驟

    1. 由主選單進入「設備資料建檔」。
    2. 查詢設備清單，新增或編輯設備資料。
    3. 點選「儲存」完成。

    MCR130｜型號工序建檔

    • 對應畫面： https://s3.hicloud.net.tw/leekuo/HTML/image/mcr130.png

    標準操作步驟

    1. 由主選單進入「型號工序建檔」。
    2. 查詢型號後新增/編輯工序。
    3. 點選「儲存」完成。

    標準操作步驟

    2. 查詢後新增或編輯合約內容。
    3. 點選「儲存」完成。

    標準操作步驟

    1. 由主選單進入「維護案件設備建檔」。
    2. 查詢案件後新增或編輯設備關聯。
    3. 點選「儲存」完成。

    2. 查詢設備後新增或編輯自訂工序。
    3. 點選「儲存」完成。

    MCR210｜現場人員設備工序回報

    • 對應畫面： https://s3.hicloud.net.tw/leekuo/HTML/image/mcr210.png
    • 功能目的： 現場人員回報設備工序進度並上傳照片。

    標準操作步驟

    1. 由主選單進入「現場人員設備工序回報」。
    2. 查詢後選擇工序，填寫回報內容。
    3. 需要拍照時上傳照片，點選「回報」送出。

    MCR220｜回報審核

    • 對應畫面： https://s3.hicloud.net.tw/leekuo/HTML/image/mcr220.png
    • 功能目的： 主管審核回報內容與照片。

    標準操作步驟

    1. 由主選單進入「回報審核」。
    2. 查詢待審項目並檢視明細。
    3. 點選「核准」或「退回」完成審核。

    2. 查詢條件後產出清單。
    3. 點選「下載」匯出報表。
