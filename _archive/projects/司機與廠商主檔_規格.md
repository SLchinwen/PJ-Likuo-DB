# 司機與廠商主檔｜規格

本文件定義 **司機主檔** 與 **供應商／廠商主檔**，供 CUS710 運務指派及 CUS110 出口建檔（派外車）引用。須支援**整批匯出與匯入**。

---

## 1. 司機主檔（drivers）

| 欄位 | 型別 | 說明 |
| :--- | :--- | :--- |
| driver_id | UNIQUEIDENTIFIER | PK |
| driver_name | NVARCHAR(100) | 司機姓名 |
| driver_code | VARCHAR(20) | 代碼（選填） |
| is_active | BIT | 是否啟用 |
| created_at | DATETIME2 | 建立時間 |
| updated_at | DATETIME2 | 更新時間 |

**用途**：CUS710 和群車指派；CUS110 卸送船、運輸「和群車」選司機。

---

## 2. 供應商／廠商主檔（suppliers）

| 欄位 | 型別 | 說明 |
| :--- | :--- | :--- |
| supplier_id | UNIQUEIDENTIFIER | PK |
| supplier_name | NVARCHAR(200) | 廠商名稱 |
| supplier_code | VARCHAR(20) | 代碼（選填） |
| contact_phone | NVARCHAR(20) | 聯絡電話（選填） |
| is_active | BIT | 是否啟用 |
| created_at | DATETIME2 | 建立時間 |
| updated_at | DATETIME2 | 更新時間 |

**用途**：CUS710 外派指派；CUS110 卸送船、運輸「派外車」選供應商。

---

## 3. DDL 參考

```sql
CREATE TABLE drivers (
  driver_id    UNIQUEIDENTIFIER PRIMARY KEY DEFAULT NEWID(),
  driver_name  NVARCHAR(100) NOT NULL,
  driver_code  VARCHAR(20)   NULL,
  is_active    BIT           NOT NULL DEFAULT 1,
  created_at   DATETIME2     NOT NULL DEFAULT GETUTCDATE(),
  updated_at   DATETIME2     NOT NULL DEFAULT GETUTCDATE()
);

CREATE TABLE suppliers (
  supplier_id    UNIQUEIDENTIFIER PRIMARY KEY DEFAULT NEWID(),
  supplier_name  NVARCHAR(200) NOT NULL,
  supplier_code  VARCHAR(20)   NULL,
  contact_phone  NVARCHAR(20)  NULL,
  is_active      BIT           NOT NULL DEFAULT 1,
  created_at     DATETIME2     NOT NULL DEFAULT GETUTCDATE(),
  updated_at     DATETIME2     NOT NULL DEFAULT GETUTCDATE()
);
```

---

## 4. 匯出／匯入

- **匯出**：Excel 或 CSV；欄位對應見上表
- **匯入**：整批匯入；重複檢查依 supplier_code / driver_code 或名稱
