# 02_Analysis（分析期）

本目錄用於存放**分析期**產出，與 [GOV_e化專案治理總則](../GOV_e化專案治理總則.md) §2.1 對齊。

## 用途

| 產出類型 | 說明 |
| :--- | :--- |
| RFP／系統全貌 | 系統目標、階段劃分、主作業分類、主程式清單 |
| 主程式架構 | 主程式代號、程式名稱、所屬作業、簡要說明；工程師先取得 |
| PRD（分開） | 每主程式＋子程式一份；四項必備 |
| 流程圖、To-Be 設計 | Mermaid 流程、瓶頸分析、情境說明 |
| 雛型、操作步驟 | 雛型 HTML、操作步驟說明（有 UI）；或批次規格（無 UI） |
| 驗收條件、需求變更 | 結構化驗收、需求變更單、客戶回饋進版 |

## 命名建議

依治理總則：`YYYYMMDD_類型_關鍵字.md`（例：`20260223_PRD_CUS110出口建檔.md`）。雛型檔名見 [雛型產出規範與指引](../internal_docs/雛型產出規範與指引.md)。

## 生命週期

本階段產出達 **PRD 確認＋雛型／規格定稿** 後，產出**交付開發包**釋出給工程團隊。流程見 [需求到交付包｜生命週期與流程總覽](../internal_docs/需求到交付包_生命週期與流程總覽.md)。
