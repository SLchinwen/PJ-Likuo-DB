# 功能與說明

本頁提供**程式清單**與**雛型檢視**連結，供客戶確認功能與操作流程。雛型為半成品模擬，正式功能以實際上線系統為準。

---

## 命名規則說明（對外雛型）

- **對外雛型檔名**：一律用**程式代號 + 程式名稱**命名，**檔名不分版次**；格式為 `<程式代號>_<程式名稱英文或縮寫>.html`（例：`CUS110_ExportCase.html`）。生成對外文件時將**最新版**更新至 MkDocs，覆蓋或維持此單一檔名。
- **版次呈現**：版次、更新日期不寫在檔名，改於雛型 **HTML 內容或標題**（如頁首、`<title>`）呈現。
- **避免亂碼**：檔名僅使用英文、數字、底線；程式名稱若為中文，檔名請用英文縮寫或拼音。表格內「程式名稱」「用途」等顯示文字可使用中文。
