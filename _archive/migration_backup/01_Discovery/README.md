# 01_Discovery（診斷期）

本目錄用於存放**需求診斷期**產出，與 [GOV_e化專案治理總則](../GOV_e化專案治理總則.md) §2.1 對齊。

## 用途

| 產出類型 | 說明 |
| :--- | :--- |
| 訪談紀錄 | 客戶訪談筆記、現況 (As-Is)、痛點整理 |
| 客戶資料卡／公司背景 | 客戶與專案背景摘要（可依 `00_Templates/Tpl_客戶與專案背景.md` 產出） |
| 需求展開 | 需求清單、優先順序、來源標註 |

## 命名建議

依治理總則：`YYYYMMDD_類型_關鍵字.md`（例：`20260223_訪談_出口作業現況.md`）。

## 生命週期

本階段產出為 **RFP／系統全貌** 與 **02_Analysis** 分析之輸入。流程見 [需求到交付包｜生命週期與流程總覽](../internal_docs/需求到交付包_生命週期與流程總覽.md)。
