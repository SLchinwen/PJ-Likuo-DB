# 中央標準｜需求釐清與交付包｜治理規範

本規範定義**需求釐清階段**與**產出交付開發包時**須補足之項目，確保 AI 與工程團隊有完整規格可依循。與 [docs_治理規範](../docs_治理規範.md) §6、[子程式清單與實作建議｜治理規範](子程式清單與實作建議_治理規範.md) 併用。

---

## 1. 欄位名稱對照表（必備）

### 1.1 需求釐清時補足

| 時機 | 要求 |
| :--- | :--- |
| **需求釐清** | 產出或更新 PRD／需求文件時，**必須同時**建立或更新**欄位名稱對照表** |
| **交付包產出** | 有 UI 之主程式交付包**必須**含 `01_規格/欄位名稱對照表.md`；若缺漏，AI 應**自動產出**或**提醒補足** |

### 1.2 欄位對照表內容

- 涵蓋該程式**所有表單、查詢條件、表列、開窗**之欄位
- 每欄：中文名稱、英文、縮寫（若有）、用途、所屬表單／作業
- **建議含**：**必填／選填**、**格式／長度**（依 PRD 或雛型）、**錯誤情境**（如統編格式錯誤、權限不足時顯示）；若 PRD 或雛型有定義則產出時納入，利於驗證與驗收
- 主檔引用時（如司機、廠商）註明來源主檔

### 1.3 檢查與產出

產出交付包時，依 [AI_產出交付文件指引](../AI_產出交付文件指引.md) Step 4 產出 `01_規格/欄位名稱對照表.md`。若該程式尚未有對照表，AI 應依**雛型、PRD、操作步驟說明**產出並納入交付包。

---

## 2. 主檔匯出與匯入（單檔主檔類）

### 2.1 適用範圍

下列**單檔主檔類**程式，須規劃**整批匯出與匯入**功能，方便資料移轉與環境同步：

| 主檔類型 | 範例 | 匯出／匯入 |
| :--- | :--- | :--- |
| 公司／客戶主檔 | CUS010 客戶主檔 | 整批匯出、整批匯入 |
| 聯絡人主檔 | CUS010 聯絡人 | 可併入客戶主檔或獨立 |
| 司機主檔 | 運務指派用司機名單 | 整批匯出、整批匯入 |
| 供應商／廠商主檔 | 派外車供應商 | 整批匯出、整批匯入 |
| 其他單檔主檔 | 依專案定義 | 同上 |

### 2.2 需求與 PRD 說明

撰寫 PRD 或需求文件時，若涉及上述主檔，須於**功能列表**或**非功能需求**中註明：

> 本主檔須支援**整批匯出**（Excel／CSV）與**整批匯入**，供資料移轉、環境同步與備份使用。其他程式有共用需求時可引用。

### 2.3 共用主檔引用

當其他程式（如 CUS710 運務查詢）需引用主檔（司機、廠商）時：

- 於該程式之**欄位對照表**註明主檔來源
- 於**PRD 或子程式清單**註明「引用 XXX 主檔」
- 主檔程式須先或同步開發，或約定 API／TABLE 介面

## 3. 跨程式資料來源說明

### 3.1 適用情境

當程式 A 之**資料來源**來自程式 B（或 B 所維護之 TABLE）時，須於需求或 PRD 中**明確說明**。

**範例**：CUS710 運務查詢作業之資料來自各主作業（出口、進口等）建檔內填寫之運務需求。

### 3.2 PRD／需求文件必備章節

涉及跨程式資料來源時，PRD 或需求文件須含下列章節（或等效說明）：

```markdown
## 資料來源與關聯

| 本程式欄位／資料 | 來源 | 說明 |
| :--- | :--- | :--- |
| 和群單號、客戶、卸送運等 | CUS110 出口建檔、CUS210 進口建檔… | 各主作業建檔內之運務需求（簽單、運輸區塊） |
| 司機清單 | 司機主檔 | 和群車指派用 |
| 廠商清單 | 供應商主檔 | 外派指派用 |
```

### 3.3 說明要點

| 要點 | 說明 |
| :--- | :--- |
| **來源程式** | 資料來自哪支主程式或哪個 TABLE |
| **關聯欄位** | 本程式欄位與來源欄位之對應 |
| **產生時機** | 資料何時產生（建檔時、成單時、結案時等） |
| **依賴關係** | 來源程式須先或同步開發 |

### 3.4 交付包內引用

交付包之 `PRD_本階段範圍摘要.md` 或 `欄位名稱對照表.md` 可簡要引用「資料來源與關聯」，並指向完整需求文件。

## 4. 產出交付包時之檢查清單

產出或組裝交付開發包時，除 [子程式清單與實作建議｜治理規範](子程式清單與實作建議_治理規範.md) 之檢查外，須完成：

| 檢查項 | 說明 | 若未通過 |
| :--- | :--- | :--- |
| **欄位名稱對照表** | `01_規格/欄位名稱對照表.md` 已產出 | AI 自動產出或提醒補足 |
| **主檔匯出入** | 若為單檔主檔類，PRD／需求已註明匯出匯入 | 提醒補註於需求 |
| **資料來源說明** | 若為跨程式資料，PRD 已含「資料來源與關聯」 | 提醒補註於 PRD |
| **工程師開工指引** | `03_附錄/工程師開工指引.md` 已產出 | 依 [工程師開工指引｜治理規範](工程師開工指引_治理規範.md) 產出 |
