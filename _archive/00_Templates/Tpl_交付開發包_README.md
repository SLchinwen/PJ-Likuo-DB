## 6. 使用指引

1. **開工前**：請先閱讀 `03_附錄/工程師開工指引.md`，完成「交付包接收者自檢」後再開始。
2. **規格**：以 `01_規格/` 為實作依據；欄位與 API 命名依欄位對照表與程式代號與權限矩陣。
3. **雛型**：`02_雛型/prototype/` 為定稿雛型，操作步驟見 `02_雛型/操作步驟說明.md`。
4. **取得更新**：需求變更時以需求 Repo 之新 Tag／交付包為準；本包版次與對應 Tag 見上方 §1。
