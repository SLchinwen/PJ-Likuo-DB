# 雛型標題命名標準

本標準規範雛型 HTML 頁面之**標題列**格式，確保一致性與可讀性。

---

## 1. 標題格式

### 1.1 邏輯結構

```
模組名稱 - 程式代號 程式名稱（雛形） 日期版次
```

| 組成 | 說明 | 範例 |
| :--- | :--- | :--- |
| **模組名稱** | 所屬模組或系統名稱 | 報關管理模組 |
| **程式代號** | 依程式編碼原則之主程式代號 | CUSSYS010、CUS110 |
| **程式名稱** | 該程式之中文名稱 | 表單組態維護、出口案件建檔 |
| **（雛形）** | 固定標示，表示為雛型非正式版 | （雛形） |
| **日期版次** | 版次標識，格式 `vYYYY-MM-DD` | v2026-02-23 |

### 1.2 範例

- `報關管理模組 - CUSSYS010 表單組態維護（雛形） v2026-02-23`
- `報關管理模組 - CUS110 出口案件建檔（雛形） v2026-02-12`

---

## 2. 視覺呈現

### 2.1 標題列規則

- **主標題**：模組名稱、程式代號、程式名稱、（雛形）採一般字體大小與顏色
- **日期版次**：採**藍色小字**（`color: var(--primary)` 或 `#2F80ED`，`font-size: 12px`），與主標題區隔
- **不重複**：標題內已含程式代號與程式名稱，**不再**額外顯示「程式代號｜分類｜標籤」等冗長資訊

### 2.2 CSS 建議

```css
.page-title .version-badge {
  color: var(--primary);
  font-size: 12px;
  font-weight: 500;
  margin-left: 8px;
}
```

### 2.3 HTML 結構

```html
<div class="page-title">
  報關管理模組 - CUSSYS010 表單組態維護（雛形）<span class="version-badge">v2026-02-23</span>
  <!-- 其他按鈕等 -->
</div>
```
